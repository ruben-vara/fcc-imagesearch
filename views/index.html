Image Search Abstraction Layer

This app is a image search engine, created as a freeCodeCamp project:

  1. Write the searched word as a parameter, after /api/search/.
  2. Add ?offset=20 at the end, or any other number, to paginate throguh the results.
  3. Also, if you visit this link, you can view the 10 latest searched words.

Cool, huh?

You can check it easily, here:

Go to this url and see the JSON output.

https://majestic-pump.glitch.me/api/search/dog?offset=32

Or visit this url to check the latest search terms:

https://majestic-pump.glitch.me/api/latest/search
